.TH killjvm 1 "October 30, 2001" "KillJVM"
.SH "NAME"
killjvm \- Terminate a JVM
.SH "SYNOPSIS"
.BI "killjvm"
.BI "["options ]
.I jvmnumber
.sp
Options:
.BI "[\-h " host ]
.BI "[\-s " service ]
.br
.SH DESCRIPTION
The program
.B killjvm
contacts a PJVM Manager, which in turn contacts its PJVM Server, and
attempts to terminate the operation of the JVM in memory specified by
.IR "jvmnumber" .
If successful,
the JVM is terminated and the port at which the JVM resided is freed.
.P
The
.I jvmnumber
is the internal identification number of a JVM registered with a PJVM
Manager.  A list of these identification numbers can be determined with
the use of
.BR "listjvm" .
.SH OPTIONS
.TP
.BI "\-h " host
The host on which a PJVM Manager
is running.  If this is a remote
server, specify a hostname.  Default is
.BR "localhost" .
.TP
.BI "\-s " service
The port number or service name at which a PJVM Manager
can be contacted on the
.B hostname
specified.  Default is
.BR "pjvmmanager" .
