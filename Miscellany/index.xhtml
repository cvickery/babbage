Miscellany

Stuff

  • Westinghouse Sign This is a re-creation of a billboard that used to be in Pittsburgh; a smaller (3-logo) version of it was outside the Midtown Tunnel to Manhattan.

    This one is a semi-random pattern, but the program is set up to be scripted … one of these days.

    The program uses the HTML “canvas” element, which is suppored by all browsers except Internet Explorer at this time. And it uses JavaScript to draw the animation.

  • Scheduling Matrix Uses the <canvas> element to show the course scheduling matrix used at Queens College (at the time it written). Click and hover everything to explore.

  • Remote Rollover Demonstrates the use of CSS for a “remote rollover” effect. Uses no JavaScript, Flash, or whatever. Just CSS.

  • Another CSS Demo. This one demonstrates how to disguise a checkbox so it looks like a button, and using its checked/unchecked state to control the visibility of another element on the page.
  • Application to demonstrate some of the principles of audio sampling.
  • Fourier Synthesis: Build your own complex waveform: explore analog parameters such as frequency and phase, and digital sampling parameters as well.
  • Boolean Minimization: tutorial on boolean minimization (algebraic, Karnaugh Map, and Quine-McClusky) with a program to do the work for you.
  • Generate ID JavaScript application that uses an anonymous self-executing function and a static local variable to generate a sequence of numbers with zero global variables.
  • Student David Edell’s Tetris Game using Celoxica’s PixelStreams and Handel-C.
  • Student Eric Shamow’s Persistent Java Virtual Machine project, an exercise using the Java reflection mechanisms.
  • Landscape Contours: a web page that generates a table dynamically using JavaScript. The page is XHTML 1.1 served as application/xhtml+xml if the user’s browser will accept that media type. Both the XHTML and CSS validate properly. Like everything else here, it uses PHP to generate the correct MIME type header and to put the Last Updated timestamp at the bottom.

Utilities

Scripts and small programs I have used regularly in the past. Most of the links are to code listings that can be cut and pasted into a text editor.

mkhtml
Bash script that generates web pages from source code files. Used to generate the web pages for the three utilities given here.
pmw
Bash script to gives the word read access to its arguments. Recurses through directories. Requires Cygwin and the file command to run on Windows.
chkformat
A C program to check whether file arguments contain tabs and/or lines wider than a given width (default: 80). Output is suitable for pasting into grading reports for student projects.
