C and UNIX Shell Programming

CS-381.2 DDD1 (Registration Code 3292)

Tuesday and Friday 11:00 - 11:50 AM.

2 hr., 2 cr. Prereq.: CS-203.

Description

This is an experimental course that will introduce programmers to conventional C programming, with an emphasis on pointers and memory management. It does not cover object-oriented programming, and it does not assume a knowledge of Java. (This may be interesting, because your professor will be teaching a Java course this semester, too.)

The course will also introduce you to shell programming using the KornShell. There will be projects that tie the two subjects together.

Textbooks

  • Reek, Kenneth A. Pointers on C. Addison-Wesley, 1998. ISBN 0-673-99986-6.
  • Vickery, Christopher. UNIX Shell Programmer's Interactive Workbook.. Prentice Hall, 1999. ISBN 0-13-020064-6.
  • Harbinson, S. P. and Steele Jr., Guy L. C: A Reference Manual, Fourth Edition. Prentice Hall 1995. ISBN 0-13-326224-3
