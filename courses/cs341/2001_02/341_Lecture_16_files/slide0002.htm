Combinational Building Blocks
• Decoder
–  n inputs, 2n outputs
– Exactly one output true at a time, telling what
combination of inputs is true
• Multiplexer
–  n control inputs, 2n data inputs, one output
– “Connect” one input to the output
– Can implement any function of n-1 variables
• Half/Full Adder
– Two data inputs; sum and carry outputs
– Full adder has carry in
