Types of Gates
Buffer, Inverter
One input, one output
Buffers are useful when you need a restorer, but no combiner.
AND, OR, XOR
Two or more inputs, one output.
XOR can be constructed from AND, OR, and Inverters.
NAND, NOR, XNOR
Two or more inputs, one output.
NAND and NOR are universal; all of the other gate types can be built using any one of these types.

Boolean Functions
Sixteen functions of two variables.
Each 4-bit number corresponds to the output column of a truth table of two variables
