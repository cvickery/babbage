Executing an IJVM add
Instruction
• This occurs after an add instruction’s op code has
been fetched (read) from memory.
– Copy TOS (which holds a copy of the value in the top
of stack location in memory) to H
– Subtract 1 from SP to get the address for reading the
second from top of stack memory location.
• MAR, SP ç SP – 1
– Read the second from top of stack memory location
into the MDR.
– MDR, TOS ç MDR + H
– Write the value in the MDR to the new top of stack
location in memory.
