Error Detection and Correction
• Section 2.2.4 of the textbook.
• Used when transmitting binary data over a “noisy
channel.”
• Basic principle is to add redundant information (reducing
effective bandwidth).
– Transmitter adds the extra bits.
– Receiver uses the extra bits to check the integrity of the data
received.
• Detection: Receiver can recognize invalid codewords when
they are received.
– Signal an error or request transmitter to send data again.
• Correction: Enough redundant information is sent so the
receiver can determine which bit(s) are in error.
– Correct by inverting the error bits.
