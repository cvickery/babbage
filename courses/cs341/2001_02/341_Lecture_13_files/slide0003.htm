Properties of Gates
• Propagation Delay
– The time it takes for the output of a gate to change state in
response to a change in the input(s).
– Typical value: 1 nanosecond.
• Fan-in
– The number of wires going into a gate
• Fan-out
– The number of inputs to which the output of a gate can be
connected without overloading the ability of the restorer to
generate logically valid voltages.
– Can be determined only by reading the manufacturer’s “data sheet”
for the gate.
– Circuits that violate the fan-out limits produce undefined values.
