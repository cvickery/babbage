Logic Networks
“When one gate is not enough.”
• A combinational logic network is a mechanism
for producing a boolean output value based on
the values of a set of boolean input variables.
• A combinational logic network may be
completely specified in any of three equivalent
representations:
1. A boolean equation.
Y = f(a, b, c)
2. A truth table showing the output value for every
combination of input values.
3. An electronic circuit constructed from gates.
– This representation is the ultimate goal in logic design.
