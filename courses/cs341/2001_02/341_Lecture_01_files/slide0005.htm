Clocks and State Changes
w A clock is a signal that repeats at regular intervals.
n The repetition rate is measured in Hertz (Hz).
n The repetition interval is the reciprocal of the rate, and is
measured in seconds.
n Use powers of ten for measuring these.
l 1 MHz = 1 * 106 = 1,000,000 pulses per second
l 1 msec = 1 * 10-3 = 0.001 seconds per pulse
w Each clock pulse causes the CPU to change its
state.
n Datapath cycle: Connect two registers to the ALU,
perform an operation, and return the result to a register.
n All in one clock cycle.
