Encoding Text: Unicode
• 16 bits per character
• First 256 codes (0x0000 through 0x00FF) are the
same as ISO Latin-1.
• Used by Java, Windows NT, …
• Version 3.0.1 of the standard has 49,194 different
characters.
– ISO/IEC 10646: ISO name for the standard.
– Karit, Manipuri, Moso, Pahawh Hmong, Rong, Tai Lu,
Tai Mau, and Tifinagh are not yet supported!
• http://www.unicode.org has information about the
standard, the consortium that is developing it, etc.
