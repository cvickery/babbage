Questions

  1. Why is the machine in the video named “Frances?”
  2. Look at a few of the other Galton Boards on YouTube. Many of them are not very “good.” How would you decide whether a board is “good” or not? Why do you think it's hard to build a good one?
  3. How many bins would there be in a Galton Board with 7 rows of pins?
  4. A Galton Board is not a binary tree. How many bins would there be in a binary tree with seven rows?
  5. Draw a Galton board with 5 rows and a binary tree with 5 rows.
  6. Number the bins of a 5-row Galton board left-to-right, starting at 0.   7. The probability that a ball will end up in a particular bin is the percentage of balls that end up in that bin, divided by 100. For example, if 75% of the balls end up in Bin 1, the probability is 0.75.

    Since all the balls end up in some one of the bins, the sum of the percentages for all the bins is 100%, and the sum of the probabilities is thus 1.0.

    Add a column to the six-row table you built in Question 6 and fill in the probability a ball will end up in each bin. Check your work: the sum of the probabilities has to be 1.0. (Take the answers to 5 decimal places and there will be no rounding errors.)

  8. Draw a bar graph that shows the probability of a ball falling into each bin. Be sure to label each axis and to include the proper scale values. The Y axis should be drawn so the heights of the bars match the probabilities for each bin. This sort of graph is called a probability mass function.

    Does it look like a “bell curve” to you?
