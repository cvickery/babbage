Written Exercises

  1. What feature of African languages made it so difficult for Europeans to understand how drumming languages work? How many symbols do drumming languages use, and what are those symbols? What information cannot be encoded using those symbols, and how do the drumming languages communicate the information that cannot be encoded using those symbols?

    Write your answers to the above questions as a short essay. That is, instead of simply writing answers to each question separately, structure your answers as a coherent description of how drumming languages work and how they relate to other languages. Your essay should be 2-3 paragraphs long. Do not quote Gleick directly, but cite his work in your essay. Use any citation format you are familiar with.

  2. How many symbols does Morse Code use, and what are they? How many symbols are there per character, and how many per word? You can’t answer those questions with numbers, so how would you go about answering them? In your answer, include an explanation of the Morse Code tree diagram distributed in class on October 5.
  3. How many symbols do binary codes use to represent information? How many binary symbols are used to represent a character using the ASCII code? Using ISO-Latin-1? Using Unicode? To represent sound pressure level on an audio CD?
  4. Make up a binary code to represent the classic colors of the rainbow: red, orange, yellow, green, blue, indigo and violet. What is the theoretical minimum number of bits you would have needed for each color, and why did you have to use more?

    What you needed to do was to assign binary numbers to colors. Now assign different binary numbers to the same seven colors. That is, make up a second code.

    Computer displays typically display colored dots (pixels) as combinations of red, green, and blue lights of different intensities. The intensity levels vary from 0 (no light) to 255 (maximum amount of light) for each color channel. On the web, rgb(0,0,0) represents the color with the red, green, and blue channels all set to the minimm, which is black. rgb(255,0,0) is a mix of maximum red, no green, and no blue, which is a pure red. rgb(255,255,0) is maximum red and green, and no blue, which is yellow.

    How many bits are needed to represent all the values from 0 to 255?

    Write an expression using exponents, multiplication, and addition that gives the total number of different colors that a pixel can have.

    What rgb values would produce a white pixel?

    What color would rgb(127,127,127) represent?
