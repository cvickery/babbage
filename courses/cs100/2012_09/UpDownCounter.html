Title:
Up/Down Counter
Description:
5-bit binary counter counts up when IN_2 is pressed; counts down when IN_3 is pressed. Out_E is lsb; Out_A is msb. Uses Var_9 to poll if either input is true.
Inputs:
    In_1:

    In_2:
count up

    In_3:
count down

    In_4:

    In_5:

Outputs:
    Out_A:
16's output

    Out_B:
8's output

    Out_C:
4's output

    Out_D:
2's output

    Out_E:
1's output
