Printing
• ( print “Hello, World!” )
• ( format <print_it?> <format_string>
    <value> … )
– If <print_it?> is true, the formatted string is printed and
format returns nil
– Otherwise, format returns the formatted string as its
result.
– (print (format t “hello” ))
– (format t “~(~A~)~%~a” ‘hello “there”)
