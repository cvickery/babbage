Tail Recursion vs Iteration
If the last form evaluated in a function is a recursive call, it’s tail recursion.  Most lisp implementations will optimize this code so intermediate stack frames are not kept.
[ TailCount.lsp ]
(DOTIMES (var count [result]) body)
(dotimes (i 5 "Done") (print i)))
(DOLIST (var list [result]) body)
(dolist (x ‘(1 2 3) “Done”) (format t “The square of ~a is ~a.~%” x (* x x)))
(DO (var [init [rep]]…) (test results …) body)
(do (( i 0 (+ i 1))) ((= i 5) "done") (print i))
