List Manipulations
CONS vs LIST
CONS takes two args, LIST takes any number
LIST appends NIL to last CDR
(NTHCDR n list)
(nthcdr 2 ‘(a b c) )
(LAST list)
Returns last CONS cell of a list, which is a list.
(LENGTH element)
Element can be a list or a string

Function MY-LAST
(defun MY-LAST (list)
( nthcdr (- (length list) 1) list ))

More List Utilities
(REVERSE sequence)
Sequence can be a list or a string
(APPEND list list …)
Concatenates lists
(defun pal (list)
(list list (reverse list) ))
(elt sequence index)
(elt ‘(a b c) 1)
(elt “Why a duck?” 0)

Simple Predicates
LISTP
NUMBERP
ATOM
Anything other than a cons cell

Equality
Eq
Most strict.  Must be same location in memory.
Two different cons cells will not be eq
Eql
Big Numbers will test eql if they have the same value.
Equal
True if the two items look the same.
Equalp
Most relaxed.  True if they have same numerical value, same characters, ignoring case, etc.
