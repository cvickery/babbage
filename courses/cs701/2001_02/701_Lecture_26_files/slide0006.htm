Escape Sequences
• Arrow keys, which are to be ignored, and the delete key,
which is to be treated as backspace and rubout are, are both
examples of “escape sequences.”
– If you’re delete key doesn’t generate the escape sequence given in
the Assignment 5 web page, you can still test your program by
typing each of the characters in the sequence yourself.
• Highly recommended but not required is to write a
function that handles escape sequences in a general way.
See Assignment 5 web page for more information.
– You should know how to work with C’s enum feature whether you
write this function or not.

– Program that uses the recognize_escape() function
