File Access Flags
 The file access flags are as follows:
  O_RDONLY
      The file is open for reading only.
  O_WRONLY
      The file is open for writing only.
  O_RDWR
      The file is open for reading and writing.
  Exactly one of the file access values (O_RDONLY, O_WRONLY, or
O_RDWR)   must be specified.  If none is set, O_RDONLY is assumed.
