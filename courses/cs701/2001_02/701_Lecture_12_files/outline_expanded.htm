 make and Multiple Modules
The (g)make program has built-in rules for converting files from one format to another based on the filename extensions.  For example:
  .c.o :
  <tab> $(CC) $(CPPFLAGS) $(CFLAGS) –c $<
But make doesn’t have any built-in rules for linking multiple object modules to produce an executable file.
