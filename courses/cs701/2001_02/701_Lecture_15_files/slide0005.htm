Checking a File’s Status
char *pathname = “/usr/bin/ls;
struct stat statBuf;
   stat( char* pathname, &statBuf )
– Returns zero if file exists.  statBuf will contain lots of
information about the file, such as size, time/date of last
modification, etc.
• But for this assignment, all we need to know is that it exists;
execve() will check that it is executable, etc.
