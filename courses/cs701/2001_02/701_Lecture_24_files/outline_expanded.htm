Aliases
Definition
 alias Name = Value
 alias is the name of a new builtin command.
Name and value must be tokens. Value might be a string token.
Substitution
While parsing (see below) see if the first token on a command line is an alias Name.  If so, substitute Value for it.
Need to tokenize Value in case it was a string with spaces in it.
