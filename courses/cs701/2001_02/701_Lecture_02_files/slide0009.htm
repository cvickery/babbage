The File System
• “Everything is a file” in Unix, even if it isn’t
really.
• There is just one root node ( / )
– Multiple volumes may appear within the tree
• E.g., /mnt/floppy might be the path to the root of your floppy
disk, unlike the “A: drive” concept of DOS/Windows.
– Device drivers (code in the kernel), processes, and
sockets are all examples of non-files that look like files
in a Unix file system.
