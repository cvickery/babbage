Library Calls
• To save programmers the work of writing the
same utility functions over an over, there is are
library files that contain pre-compiled functions
that applications can call.
– Every C program is automatically linked to the
Standard C Library, but others can be added.
– Some library functions also make kernel calls, but they
minimize the number of kernel calls as much as
possible.
• Example: The only kernel call that does output is
write().  The library function fwrite() calls write(),
but not necessarily every time it is called.
