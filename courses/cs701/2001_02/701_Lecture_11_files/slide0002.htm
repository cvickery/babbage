Multiple Modules
• The linker, ld, has the job of resolving symbolic references
between program modules.
–  main.cc references sh_exit()
–  sh_exit.cc defines sh_exit()
– Linker puts the address of the first byte of sh_exit() into the
dispatch table.
– Use a header file to make sure the definition matches the reference
with respect to return type, number of arguments, and types of
arguments:
// sh_builtins.h
#ifndef __SH_BUILTINS__
#define __SH_BUILTINS__
typedef int bfun( int, char *[] );    // Built-in function type
extern bfun sh_exit;
#endif
