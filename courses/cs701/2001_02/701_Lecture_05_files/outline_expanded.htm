Make and Environment Variables
CC – What compiler driver to use
 cc is the default; we want gcc
CFLAGS – What options to specify on the compiler driver command line
We want –g and –Wall
You can add the setenv commands to your ~/.cshrc file so they will be executed every time you log in.

FAQ
“Why do I have to specify the full pathname to my executable file to get it to run?”
Linux systems in particular do not include “.” in your PATH by default.  But you can do so in your ~/.profile (or ~/.cshrc).
It’s a security feature to deter hackers.
