More About g++
On forbin, you have to use gmake instead of make to get automatic recognition of .cc files as C++ instead of C code.
Contrary to what I said in the previous lecture, C++ has literal constants true and false that you can assign to variables of type bool.
Non-zero integers still are treated as true and 0 is still treated as false.
