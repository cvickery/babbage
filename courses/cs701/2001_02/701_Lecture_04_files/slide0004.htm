The Make Utility
• Command: make <target>
• Built-in Rules
– Based on file suffixes.  It knows how to convert .c files
into executable files, but doesn’t know how to make .c
files, because people do that using an editor.
• Environment Variables
– You have to set the CC and CFLAGS environment
variables to get make to generate the correct gcc
command for you.
•  setenv CC gcc
•  setenv CFLAGS “-g –Wall”
• More complicated rules
– Use a Makefile
