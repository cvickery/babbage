Algorithm: sh_exit()
• Prototype
 int sh_exit( int argc, char *argv[] );
• If argc is one, call exit( 0 )
• If argc is two, convert argv[ 1 ] from a string to an
int, n, and call exit(n)
– Pedantic: print an error message if argv[1] is not a
numeric string.
– Use strtoul() to do the conversion.
• Otherwise, write an error message to stderr, and
return –1.
