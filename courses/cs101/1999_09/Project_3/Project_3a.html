From the directory containing your LibraryManager.java file, give this command: 
      jar cvf project3a.jar LibraryManager.java
Then, just attach the file named project3a.jar to your email message, and send it to me.

Description

All you need to do is to write an application that has a working exit button and which also exits if the user clicks on the "x" button in the upper right corner of the Frame. Use javax.swing classes JFrame, and JButton, as described in class.

You do not have to make your program run as both an applet and as an application, but I encourage you to experiment with this feature once you have the basic application working.

You do not have to make your application's Frame come up centered in the user's screen, but I encourage you to experiment with this feature once you have the basic application working.

You do not have to make your Exit button have a tool tip, but I encourage you to experiment with this feature once you have the basic application working.
