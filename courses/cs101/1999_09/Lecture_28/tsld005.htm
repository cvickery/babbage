Layout Managers

  • Calculates the geometry of the components in a container.

    • Always use layout managers instead of absolute coordinates.

  • FlowLayout (default for JPanel ), BorderLayout (default for content panes of JFrame and JApplet), Grid, Gridbag, TabbedPane, etc.

  • Put containers in containers.

    • Each gets its own layout manager.
