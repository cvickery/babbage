Model - View Architecture

  • A Data Model provides access to some data.

    • May use containers, such as arrays or Vectors
    • Might access a remote database to get the data.
    • An interface provides a controlled way to use the model.

  • A Component provides a way to view (look at) some data.

    • Swing components have data models they know how to work with.
