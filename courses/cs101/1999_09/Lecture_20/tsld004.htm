Character Codes

  • ASCII: 7 bits per character
    • Used in the US.
  • ISO-Latin-1 (8859-1): 8 bits per character.
    • When leftmost bit is zero, is same as ASCII.
    • Used where languages are based on the Latin alphabet, but have accent marks.
  • Unicode: 16 bits per character.
    • When leftmost 8 bits are zero, same as ISO-Latin-1.
    • Used for Java char values.
    • By default, BufferedWriter.write() converts char variables in memory to ISO-Latin-1 bytes on disk.
    • By default, BufferedReader.read() and BufferedReader.readLine() convert Latin-1 bytes to Unicode characters.
