Classes String and StringBuffer

  • Both String and StringBuffer, internally, hold a string as an array of char.
  • String objects are immutable; StringBuffer objects can be modified.
  • String methods (about 60 of them)
    • charAt, compareTo, compareToIgnoreCase, getChars, getBytes, lastIndexOf, startsWith, endsWith, replace, length, ...
  • StringBuffer methods (about 35)
    • append, charAt, delete, deleteCharAt, insert, length, replace, reverse, subString, ...
