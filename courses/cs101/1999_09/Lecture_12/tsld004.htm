Getters and Setters

  • Public methods that provide access to private, protected, or package fields.

  • If xxx is the name of a field:

    • Read Access
      • type getXxx()
      • boolean isXxx()
    • Write Access
      • setXxx( type )
      • resetXxx() or clearXxx()

  • The “Java Beans” naming convention.

    • The java.lang.Class reflection methods.
    • The javap command.

  • The Principle of Encapsulation.
