Subclasses Inherit Fields

  • Any field defined in a class can be accessed from a method defined in a subclass.

    • Except private fields
    • Protected fields can be accessed only from subclasses (and the class where they are defined). This is more restrictive than package access
    • Final fields’ values can’t be changed by anybody.
    • You can hide a superclass’ field by redefining it in a subclass, but there is no reason to do so.
