Controlling Access to Fields and Methods

  • Public ( public )

    • Any method in any class can access it.

  • Private ( private )

    • Only methods in the same class can access it.

  • Protected ( protected )

    • Methods in the same class or its subclasses can access it.

  • Package (nothing specified)

    • Methods in classes in the same directory can access it.
