Inheritance

  • A way of reusing the code in existing classes by adding new features to them.

    • Example: class Applet was written and debugged by Sun. When you extend this class you use all the code that Sun wrote, and add your own features to make the Applet do what you want.
    • Example: class MP3header was written and debugged by Dr. Vickery. If you wanted to change the bit rate of an MP3 file, you could extend my class and add a method to do that.
