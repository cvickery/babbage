Methods Can Be Overridden

  • If a subclass defines a method that has the same signature as a method in its superclass, the subclass’s definition overrides the superclass’s definition.

    • The subclass’s method can invoke the superclass’s method by using super.xxx()
    • But a class can declare a method final, which means it can’t be overridden in any subclass.
