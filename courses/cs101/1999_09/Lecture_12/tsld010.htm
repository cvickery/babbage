Subclasses Inherit Methods

  • Example: Class Object is a superclass of all other classes in Java. It has methods named clone(), finalize(), getClass(), and toString() (among others). So you can use these methods with objects of any class.

  • Example: Class ID3V1info has a showInfo() method, so any subclass of ID3V1info automatically has that method available.
