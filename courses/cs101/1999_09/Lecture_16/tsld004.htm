Putting a Frame on the Screen

  • Create a Frame object. The constructor takes a String for the Title bar.

    • The title can be changed dynamically by using Frame’s instance method, void setTitle( String )

  • Use Frame’s add() method to put components inside the Frame.

  • When all the components have been added to the Frame (and to any containers inside the Frame), use pack() to get their arrangement computed, and show() to put the Frame on the screen.
