The WindowListener Interface

  • Declares seven methods:

    • void windowClosing( WindowEvent )
      • Called when the user clicks the close button of a Frame.
    • void windowIconified( WindowEvent )
      • Called when the user minimizes a Frame.
    • void windowDeiconified( WindowEvent )
      • Called when the user restores a Frame.
    • Four others
      • See JDK documentation
