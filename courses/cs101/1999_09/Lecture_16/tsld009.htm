Extending an Adapter

  • If you extend an Adapter class, you can override just those methods of an Interface that you are interested in.

    class MyWindowListener extends WindowAdapter

    {

    public void windowClosing( WindowEvent we )

    {

    System.exit( 0 );

    }

  • No need to write code for all the methods in the interface, just this one.
