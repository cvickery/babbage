To Create an Object

  • Your code executes “new SomeClass()”

  • If SomeClass has not yet been loaded into memory, software called the class loader allocates memory for the class and loads the class file into memory.

    • This step is done only the first time a class is referenced.
    • All methods, whether static or not, are loaded.
    • All static fields are loaded.

  • The new operator allocates memory for the object, zeros it out, and invokes the constructor.
