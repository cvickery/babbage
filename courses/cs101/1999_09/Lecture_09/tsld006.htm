How Many?

  • Class files are loaded?

    • The one named on the java command line.
    • As many others as the program uses.

  • Copies of a method’s bytecodes are loaded?

    • One, when the class is loaded, and whether the method is static or not.

  • Copies of a class’ static fields are created?

    • One, when the class is loaded.
