Declare vs Define a Method

  • Declare -- no method body.

      public long addEmUp( long[] );

  • Define -- has a method body.

      public long addEmUp( long[] vals ) {

      long sum=0;
      for( int i=0; i< vals.length; i++)
      sum += vals[ i ];
      return sum;
      }
