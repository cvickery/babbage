Classes, Abstract Classes, and Interfaces -- slide 4/9

  • What about the method body?

    • Class
      • Must be given inside { … }
    • Abstract Class
      • May be given inside { … } but must be “ ; ” if the method is explicitly declared abstract.
    • Interface
      • Must be “ ; ”
      • Do not declare abstract because all interfaces are abstract implicitly.
