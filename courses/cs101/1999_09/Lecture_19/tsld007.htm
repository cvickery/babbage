Single Inheritance

  • Each class (except Object) is a direct subclass of exactly one superclass.

    • Easy to implement the language.
    • No ambiguity in what method is being invoked.
      • If it is defined in the subclass, invoke that one.
      • If not, invoke the one defined in the superclass.
      • If not there, go to the super-superclass.
      • Etc.
    • Polymorphism
      • e.g., toString()
