Multiple Inheritance

  • Example: A Whale is a (inherits properties from) Mammal. A Whale is a SeaDweller too. So Whale could be a subclass of both.

    • But what if class Mammal and class SeaDweller both define a method called eats()? Does myWhale.eats() invoke Mammal’s eats() or SeaDweller’s eats()?
    • In Java you would make Mammal and SeaDweller interfaces, and have the class Whale implement both interfaces to hold your whales. Only one eats() method would be defined, in class Whale, but it might appear in both interfaces.
      Class Whale implements Mammal, SeaDweller { … }
