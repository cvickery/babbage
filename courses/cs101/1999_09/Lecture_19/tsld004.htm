Abstract Class Syntax

  • Syntax to declare a class abstract explicitly:

    • abstract class someClass { … }

  • Syntax to declare a method abstract explicitly:

    • abstract void anAbstractMethod();
    • (Could have any return type and argument list.)

  • Syntax to declare a method abstract implicitly (you can do this in an interface):

    • void anAbstractMethod();
