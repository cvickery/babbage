Question 1 About My 3b Code

  • Method init() is not needed. All the code in that method can be put inside the constructor instead.

    • But init() could be useful if you wanted the program to work as both an applet and and application.
    • But init() could be useful if there was more than one constructor; they could share code by calling this method. (In this case the method name could be anything.)
