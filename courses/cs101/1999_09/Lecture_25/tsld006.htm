New Library Event Handler

  • Gets file name using JFileChooser

    • Ignore if no file selected

  • Tests if file exists

    • Displays an Error messageDialog if it does

  • Initializes the library file on disk

  • Creates a MP3_Library object for the file

  • Closes current MP3_Library if there is one

  • Enables the AddFile button
