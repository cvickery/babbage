Event Handlers

  • This is another term for callback methods.

  • In Java these methods are defined in classes that implement an EventListener interface.

    • Example: public void actionPerformed( ActionEvent)
    • This callback method is declared in the ActionListener interface.
    • The ActionListener interface is a subclass of the EventListener interface.

  • Simple event handlers should be defined in anonymous classes.
