Arrays - 1

  • Review: Primitives, Classes, Objects, and References

    • Primitive data types are defined by the Java language. They are: byte, short, int, long, float, double, char, and boolean.
    • A class is a programmer-defined data type.
      • Core classes are in packages supplied with Java
      • User-defined classes are the ones you write.
      • (But array classes are generated by the compiler.)
    • An object is an instance of a class.
    • A field (variable) holds either a primitive data type value or a reference to an object.
