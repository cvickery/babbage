Event Processing Example

  • JButton has a list of ActionListener objects.

  • The ActionListener interface declares the method public void actionPerformed( ActionEvent )

  • JButton has a public void addActionListener( ActionListener ) method that adds an ActionListener object to its list.

  • When the user clicks on the JButton, the button calls the actionPerformed() method of all registered listeners.
