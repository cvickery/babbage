Callbacks and Interfaces

  • Problem: Want to have a method called when the user interacts with a component.

    • Java doesn’t allow references to methods, only to objects.
    • Therefore the component has to be given a reference to an object, and there has to be a guarantee that the object has a method with the correct signature. If the class of the object implements an interface, the compiler can guarantee that the method will be there.
