Applet/Application

  • JApplet class provides init(), start(), stop(),and destroy() methods. You can add a JLabel to tell the user what to do. init() creates the GUI; start() and stop() can make it visible or not.

  • The application’s main() method instantiates the applet and calls its init() method to create the GUI.

  • Problem is the need to sign the applet.
