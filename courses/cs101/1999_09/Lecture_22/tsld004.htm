FMQ (Frequently Missed Questions)

  • An adapter class is a class that provides default implementations for all the methods in an interface.

  • The fact that applets can be run from web pages is not a benefit of OO programming. Code reuse, late binding of methods to objects, and encapsulation are.

  • The canonical form of C:My Projects\Project_2\Docs\..\.\..\Project_1 is C:\My Projects\Project_1
