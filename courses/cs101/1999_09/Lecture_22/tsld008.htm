imports

  • import java.awt.*;

    • For Layout managers, Dimension, Point, etc.

  • import java.awt.event.*;

    • For event handling -- same in Java 1 and Java 2

  • import javax.swing.*;

    • For all visible components and containers
    • Anything you used in Java 1 gets a “J” in front of its name in Java 2.
    • Components have a “Content Pane” and a “Glass Pane” and a “Menu Bar” in Java 2.
