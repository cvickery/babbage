Classes for Byte I/O

  • FileInputStream and FileOutputStream

    • Methods read() and write() transfer (part of) an array of bytes in memory from/to a file on disk.
    • The byte array may be called a buffer.
    • If you have an array of bytes, you can convert it to a String using new String( byte[] )
    • If you have a String, you can convert it to an array of bytes using byte[] String.getBytes()
