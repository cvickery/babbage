Exam Post-Mortem - 1

  • public boolean aMethod( int[] anArray, int x );

    • There must be two parameters passed when this method is invoked, an array of int and an int. Also, the return value must be of type boolean. The only choice that passed two parameters was also the choice that used an if statement to test the true/false value returned:
      if ( aMethod( someArray, 99) ) System.exit( 0 );
