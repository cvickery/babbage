Using JOptionPane

  • Display a Dialog Box and wait for user to respond to it. (A “modal” dialog.)

    • JOptionPane.showInputDialog()
      • Show a prompt message, return a String
    • JOptionPane.showMessageDialog()
      • Show a message
    • JOptionPane.showConfirmDialog()
      • Show a message, return an int: YES_OPTION, NO_OPTION, CANCEL_OPTION, CLOSED_OPTION
