Class Exception

  • Has two constructors

    • public Exception( );
      • Also called a “no-arg” constructor because it is passed no arguments.
    • public Exception( String msg );

  • Has (among others) two methods:

    • String getMessage();
    • String toString();

  • Has many subclasses in the Core Class Libraries

    • IOException
    • etc.
