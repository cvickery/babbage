Development

  • Download and run vickery.exe -- this will give you all the classes for the project, including LibraryManager. Run the program to be sure you understand it.

  • You can download vickery_docs.exe if you want a copy of the documentation files; otherwise read them from the web

  • Write your own LibraryManager.java. The first time you compile it, you will replace my LibraryManager.class.
