Overloaded Constructors

  • Same idea as overloaded methods, just no return type.

    • Constructors may contain return statements. (What values can appear in them?)

  • A superclass’s constructor can be invoked by invoking super() as the very first statement in a constructor. super() can be overloaded, too.

  • Allows you to reuse the initialization code provided by the superclass.
