If the Invoked Method is Not Static

  • There has to be a reference to an object in order to invoke the method.

    • Object instances are created by using the new operator.

  • If the invoking method is static, you need an explicit reference to an object instance.

  • If the invoking method is not static

    • If the invoked method is defined in the same class, no explicit object reference is needed: the object used to call the invoking method is used by default.
    • If the invoked method is defined in a different class, an explicit object reference is needed.
