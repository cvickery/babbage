How Does This Apply To The Project?

  • If JFileChooser.showOpenDialog() is an instance method, so you need an object to call it.

  • JFileChooser.getSelectedFile() is also an instance method. It returns a reference to a File object.

  • MP3header.showInfo() is an instance method, so you need an object to call it.

  • MP3header’s constructor takes a reference to a File object as a parameter. Use the one returned by JFileChooser.getSelectedFile().
