JVM Memory Regions

  • Heap -- Where the objects are.

  • Method Area -- Where the instructions are.

  • Constant Pool -- Housekeeping Information

  • Program Counter (one per thread)

  • Java Stack (one per thread)

    • One frame for each method call
      • Local Variables
      • Operand Stack, including arguments and return value
      • Constant Pool references
