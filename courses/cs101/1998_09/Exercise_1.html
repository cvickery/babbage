CS-101 Exercise 1, Fall 1998

Due Date

This exercise is due on September 15, 1998.

Deliverables

Write a public class named Main which contains the static method main(). The main() method is to instantiate another public class named Exercise_1 three times. Then, main() is to call public Exercise_1's member method which_instance() twice for each instance. Each time which_instance() is called, it prints the message, "This is instance number n." with n set to the number 1, 2, or 3 as the case may be.

Use the [ Sample Code ] from [ Lecture 1 ] as a guide for how to document the code.

After testing your code to be sure it works properly, put Main.java and Exercise_1.java into one zip file, and email it to Dr. Vickery (vickery@qc.edu) by midnight on the due date. The zip file may have any name you choose, but the files in the zip file must be named exactly as indicated.

If you don't know how to create a zip file, look at [ the Development Tools web page ] for information.
