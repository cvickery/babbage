Class objects

  • A Class object is created for each class when it is loaded into the JVM’s memory.

    • Contains static fields (static primitives and static references to objects) for the class.
    • Contains pointers to static methods.
    • Is pointed to by instances of the class.
    • Contains housekeeping information used for creating new instances, including lists of all constructors, fields, and methods.
