Definition of Terms

You should be able to define, use, and understand each of the following terms:

class, interface, method, field, variable, primitive type, reference field, instance field, instance method, primitive field, static field, local field, class method and class field (same as static method and static field), final field, final method, static initializer, constructor, abstract method, abstract class, ...

public, private, protected, and package access to classes, fields, and methods.
