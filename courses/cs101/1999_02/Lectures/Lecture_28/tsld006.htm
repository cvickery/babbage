ItemEvent Processing

  • User clicks in List box, itemStateChanged() gets called.

    • Use List.getSelectedIndex() to find out which item was clicked.
    • Use ItemEvent.getStateChange() to find out if event was ItemEvent.SELECTED or not.
    • Invoke highlightItem() or unhighlightItem() to change the selected field in the display list.
    • Invoke repaint() to get the canvas to change now.
