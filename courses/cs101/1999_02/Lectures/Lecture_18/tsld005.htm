2) hide

  • processCommands() invokes “d.hide()”

    • Method hide() in class Drawing sets this.visible to false

  • processCommands() invokes “System.out.print( d )”

    • Method toString() in class CDrawing is invoked, and returns an empty String because this.visible is false. The print() method has no effect that the user can see because it prints just the empty String.
