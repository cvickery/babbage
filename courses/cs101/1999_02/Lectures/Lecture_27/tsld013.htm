Layout and Visibility

  • Window.pack() - “Causes this Window to be sized to fit the preferred size and layouts of its subcomponents.” (Quote from JDK documents)

    • “The Window will be validated after the preferredSize is calculated.” (ibid.)
    • Container.validate() - Calls doLayout().

  • Window.show() - Validate and make visible

    • Okay to use Component.setVisible( true ) if the Window is already validated.
    • Puts this window in front of others if it is already visible.
