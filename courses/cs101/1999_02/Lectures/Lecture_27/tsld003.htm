Some awt Layout Managers

  • Flow

    • Default for Panels
    • The sequence in which you add components determines their placement -- no constraints.

  • Border

    • Default for Windows
    • Constraints for five positions. Center expands.

  • Grid

    • Equal-width columns, Equal-height rows.

  • GridBag

    • Uses separate GridBagContraint objects

  • Card, ScrollPane, ...
