How Big is the Screen? A Dimension object has fields named width and height

    Dimension d;


    d = Toolkit.getDefaultToolkit.getScreenSize();

    • Can be used to position a Frame on the screen:

    Frame f = new Frame( “title” );

    f.setLocation( d.width/2, d.height/2 );

    • Puts the upper left corner of the Frame in the center of the screen.
