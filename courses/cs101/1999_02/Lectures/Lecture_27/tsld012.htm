Associate a Layout Manager with a Container

    Panel p = new Panel ( new GridLayout(3,2) );


  • or

    p.setLayout( new GridLayout( 3,2 ) );

  • or

    GridLayout g = new GridLayout( 3,2 );

    p.setLayout( g );

  • or (prevent moving or resizing components)

    p.setLayout( null );
