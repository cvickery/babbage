Class CDrawing

  • Provides toString(), which generates a printable string by traversing the display list and putting character values into a two dimensional array of char, then converting the array to a long String for printing

  • Has private methods for adding lines, circles, and rectangles to the array of char

  • Uses classes LineCoordinates and CircleCoordinates to fill in the array of char
