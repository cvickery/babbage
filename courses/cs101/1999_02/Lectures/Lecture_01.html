Running Java Programs

  1. Use a text editor such as PFE, to prepare Java Source Files (ones with .java extensions).
  2. Use javac to compile each Java source file into one or more binary Class Files (ones with a .class extension).
  3. Run a program, such as java or appletviewer, or a web browser, that simulates the Java Virtual Machine (JVM).
  4. Get the JVM to load the first class file of the program into its memory.
  5. The JVM validates the class file, links it to the other parts of the JVM already in memory, and then executes the first method (main()for applications, or init() for applets.)
