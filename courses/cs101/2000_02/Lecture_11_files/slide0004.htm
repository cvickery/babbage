Nested Method Calls
• Each time a method invokes another method (or
itself), it’s a nested call.
– The most recently called method is always the next one
to return.
– [ Nested.java ]
• main() calls methodA().
• methodA() calls methodB().
• methodB() calls methodC().
• methodC() returns to methodB().
• methodB() returns to methodA().
• methodA() returns to main().
• main() exits.
