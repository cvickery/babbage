Exercise 4

Introduction

The purpose of this exercise is for you to practice some array operations by implementing the SortedWordList class you used in Exercise 3.

Description

Write your own implementation of the SortedWordList class used in Exercise 3. The name of the class must be SortedWordList, and it must implement each of the following methods, using exactly the method signatures given here.
public boolean addWord( String word );
Searches the array to find word in one of the WordCounter objects already in the array. If the word is not found, add it to the array in its proper alphabetical position, and return true. If the word is found, add one to its counter value, and return false.
public String wordAt( int index );
Returns the word at position index in the array.
public int counterAt( int index );
Returns the value of the counter at position index in the array.
public int size();
Returns the number of WordCounter objects currently in the array.
