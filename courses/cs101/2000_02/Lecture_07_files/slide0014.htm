param, return, and throws Tags
• Put these in the comments at the beginning of each
method you define.
– @param
• One for each parameter passed to the method. Give its name,
followed by a sentence telling what it is used for.
– @return
• Tell what information is returned.  Note that the type of value
does not need to be mentioned.
– @throws
• One for each throws clause in the method signature.  Name the
type of exception that might be thrown, and under what
circumstances it would be thrown.
