java
• java <class name>
– Create a Java Virtual Machine (JVM).
• A program that simulates a hypothetical CPU
• There are JVMs for many different real CPUs
– Intel Pentium (PC)
– Motorola/IBM Power PC (Macintosh G3/G4))
– Compaq Alpha (forbin)
– Load the class file named on the command line
into the JVM’s memory.
– Invoke the main() method in that class.
