Setting UP PFE
• Use Options->Preferences to set up:
– Associations
– Backup Policy
– Others you may be interested in
• Use Options->Default Modes to set up:
– Edit Mode Groups (Make one for Java Files)
– Edit Modes to set:
• Input (auto-indent, strip trailing blanks)
• Language Awareness (use “C”)
• Text Formatting (Use “Soft Tabs” every 2, 3, or 4 characters;
enable text wrapping at column 72 so you can see when lines
are too long.)
