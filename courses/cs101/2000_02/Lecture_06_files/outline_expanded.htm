Exercise 4 Concepts
When the SortedWordList is created, set up an array of WordCounter references with room for 10 elements.  Set the private fields size and capacity to zero and 10.
Each time a word has to be added to the list, add 1 to size until it exceeds capacity.  When this happens, create a new array with double the capacity of the old one, and copy the old one into the new one.
When you add a word into the middle of the array, you have to move other elements out of the way.  Watch out for inserting at the beginning of an array.
