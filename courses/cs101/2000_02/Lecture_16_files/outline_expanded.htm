Users, Clients, Developers, and You
User
The person who interacts with a program while it is running.
Client
A program that uses a class that was written by someone else.
Developer
A person who writes programs for users and/or other developers.
You
All three of the above!

@throws, throws, and throw
@throws
Javadoc tag telling the name of an exception type and when the method might throw an exception of that type.
throws
A keyword that follows the name of a method, and which is followed by all checked exception types the method might throw.
There should be a @throws tag for each exception type listed.
throw
An operator that causes an exception to be thrown.
Unless it is inside a try block, it causes abrupt termination of the method that is executing.
