JCheckBox.addItemListener( ItemListener );
The ItemListener interface requires one method:
public void itemStateChanged( ItemEvent );
Called for every click in the checkbox.
To determine whether it was selected or deselected:
int ItemEvent.getItemSelected()
Value returned will be one of these:
ItemListener.SELECTED
ItemListener.DESELECTED
boolean JCheckBox.isSelected()
Can be used any time, not only in itemStateChanged()

Introduction to Layout Managers
Containers have LayoutManagers:
Classes that determine where to place components in a container, and how big to make them.
Default LayoutManager for the content pane of a JFrame is BorderLayout.
Other containers have FlowLayout by default.
Some Others: Grid, Overlay, Gridbag

BorderLayout

BorderLayout
Zero or One Component per region.
Center grows to fill available space.
Others are sized to fit their component.
A component can be a container with its own layout manager.

GridLayout
Rows and Columns are equal height/width.
Constructor can specify number of rows and columns, or use zero for one of them to let that parameter vary.
Sequence of add() calls determines placement of components: left to right, top to bottom.
