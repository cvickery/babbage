Classes
• When you write a Java “program” you define one
or more classes.
– The java compiler generates one disk file for each class,
with a filename consisting of the class name followed
by “.class”
• In order to do anything with a class, its class file
first has to be loaded into memory.
– Transfer contents of the .class file into RAM.
– Link the class with other classes already in the Java
Virtual Machine’s RAM.
