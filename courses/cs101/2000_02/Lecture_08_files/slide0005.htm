Objects
• Once a class has been loaded, you can create
objects that are instances of the class.
– Classes may be instantiated zero, one, or multiple
times, depending on the needs of a particular program.
– Instantiation is always done using the new operator.
• Allocate memory for the object’s fields.
• Initialize the fields.
• Invoke a constructor to complete initialization.
– Compiler provides a default constructor that takes no arguments
if the class doesn’t have any constructors defined explicitly.
