Processor/Memory/Disk
• Processor interprets instructions, which are
encoded as binary numbers.
– Fetch an instruction from memory.
– Interpret (execute) it.
– Repeat previous two steps forever.
• Memory (RAM) consists of bytes, each of which
has an associated address.
• Disk is non-volatile (retains information when
turned off), and less expensive than RAM.  But
slower to access.
