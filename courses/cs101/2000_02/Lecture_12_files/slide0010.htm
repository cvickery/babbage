Java Class Hierarchy
• The root class is java.lang.Object
– All other classes are direct or indirect
subclasses of class Object
– You don’t have to declare that a class is a
subclass if it is a direct subclass of Object:
• class MyClass { … }
• class MyClass extends Object { … }
• class MyClass extends java.lang.Object { … }
• Consider class javax.swing.JApplet
