What new does, continued
• To initialize the object (assuming the class is
loaded) new does the following, in sequence:
– Allocate memory for the new object.
– Zero out all the memory for the new object.
• Boolean variables will be false, reference variables will be
null, numeric variables will be zero.
– Execute initial assignments.
– Invoke the appropriate constructor.
• There is always a constructor, even if you don’t write one.  It’s
“the default no-arg constructor” provided by the compiler.
