Initializing an Object
(what new does)
• If the class has not yet been loaded into memory,
do that before doing anything else.
• Loading a class does the following, in sequence:
– Link the class to other classes that are already loaded.
– Zero out all static variables.
– Execute static initialization statements.
– Once a class has been loaded, these steps are never
done again during the life of the JVM.
