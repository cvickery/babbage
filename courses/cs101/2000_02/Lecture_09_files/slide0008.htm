Access Control
• A class can be declared public or not.
– A public class can be used by any other class.
– A non-public class can be used by other classes in the
same package.
• A method or field can be declared public,
protected, private, or not.
– A protected member or field can be accessed only by
subclasses. (Chapter 9)
– A private member or field can be accessed only from
inside the class.
– A method or field with no access specified has package
access.
