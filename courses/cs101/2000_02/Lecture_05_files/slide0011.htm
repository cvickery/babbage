Using Class SortedWordList
• Maintains an array of WordCounter object
references.  The array may grow dynamically.
– WordCounter objects have a String to hold the word,
and an int to hold a count of how many times the word
has occurred.  Used only inside class SortedWordList.
– addWord() Adds a word (String) to the list in its proper
alphabetic position, or adds one to its counter if it is
already in the list.
– wordAt() Returns the String at a particular position in
the array.
– counterAt() Returns the int at a particular position in the
array.
– size() Returns the number of elements in the array.
