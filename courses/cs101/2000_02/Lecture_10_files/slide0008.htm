Object Based Programming
• Classes have to be loaded from disk into memory
before they can be instantiated.
– Once loaded, a class might be instantiated, zero, one, or
multiple times.
– Static variables and methods are available as soon as
the class has been loaded.
– Instance methods are loaded with the class, but can’t be
called until the class has been instantiated at least once
because you need an object to call an instance method.
– Instance variables don’t even exist until an object is
created.
