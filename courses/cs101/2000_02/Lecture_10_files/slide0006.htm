Exercise 4 (addWord()) Algorithm
• Convert the word to lowercase.
• Compare the word to each word already in the
array until:
– The word is found in the array.
• Add one to the counter and return false.
– A word is found that comes alphabetically after the
word.
• Double capacity if necessary.
• Insert the word into the array at this position and return true.
– The end of the array is reached.
• Double capacity if necessary.
• Create a new WordCounter for this word, add one to the size
of the array, add this WordCounter after the last occupied
element in the array, and return true.
