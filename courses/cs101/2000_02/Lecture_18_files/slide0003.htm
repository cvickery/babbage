GUI Development
• Need a window (JFrame) to hold the components
that make up the GUI.
• Some components are containers, which hold
other components.
– JFrame and JPanel are examples of containers.
• Other components (controls) provide the visual
elements of the graphical user interface.
– JLabel is a passive component that can display text.
– JButton is an active component that the user can
interact with (by pressing it).
