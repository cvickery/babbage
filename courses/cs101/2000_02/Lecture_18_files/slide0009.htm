Anonymous Classes
• Small classes with no name (the compiler provides
the name for you).  Defined on an as-needed basis
inside other classes.
• Example: Define a subclass of WindowAdapter
that overrides the windowClosing() method.
WindowAdapter wa = new WindowAdapter( )
{ /* class definition starts here */
    public void windowClosing( WindowEvent we )
    { /* method definition would go here */ }
 }
