Class MP3File
• Is a subclass of java.io.File
– So you can perform basic tests, etc. on it.
• Constructor takes a pathname String as its
argument.
• Has read() and write() methods.
– Analogous to TextArea’s read and write methods.
• Has ID3V1 and ID3V2 fields
– These are the “Tags”
– Each has an associated set of get/set methods.
