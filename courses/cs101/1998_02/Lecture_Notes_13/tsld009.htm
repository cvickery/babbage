Highlighting is Like Line Drawing

  • DDA Algorithm selects “pixels” (letters) to be highlighted.

    • Step along X from left to right, incrementing (or decrementing) Y by a constant each time ...
    • … unless the line is vertical.
    • Need to keep track of words that cross or overlap in case of deselecting. Keep count of how many times each letter is highlighted, then paint() highlights a letter if its count is greater than zero. highlight() adds one to the counter of each letter it highlights. unhighlight() subtracts one.
