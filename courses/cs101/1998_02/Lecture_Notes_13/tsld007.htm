Use a Hashtable to Track Highlighting

  • In WF_Frame class

    final Boolean T = new Boolean( true );

    final Boolean F = new Boolean( false );

    Hashtable ht = new Hashtable();

  • In WF_Frame constructor

    For each word, w, in the Puzzle:

    ht.put( w, F );

  • In WF_Frame.itemStateChanged()

    if ((e.getStateChange == ItemEvent.SELECTED) &&

    ( ht.get( w ) == F ) ) {

    pc.highlight( r1, c1, r2, c2);

    ht.put( w, T );

    }
