Some Hashtable Methods

  • Hashtable(int size, float loadFac)

    • size is the initial size
    • loadFac is the “load factor” -- how full it gets before automatically being resized.
    • Both parameters are optional.

  • Object put( Object k, Object e );

    • Returns previous element if k was already in the Hashtable, otherwise returns null.

  • Object get( Object k );

    • Returns the element associated with k.

  • Enumeration elements();
