Keep Track of Highlighting: Possible Solution

  • An array or Vector of boolean, one for each word in the word list.

    • Initialize to all false.
    • When the corresponding word is selected and its value is false, set its value to true and call highlight().
    • When the corresponding word is deselected and its value is true, set its value to false and call unhighlight().
