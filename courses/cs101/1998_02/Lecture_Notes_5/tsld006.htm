Using Classes: Instances and References

  • int a, b, c;

    • “int” is the data type name.
    • a, b, c are instance variables, or instances. They hold values of integers.

  • Tour x, y, z;

    • “Tour” is the class name. It is a data type name, too.
    • x, y, z are also instance variables. They can hold references to instances of Tour objects.
