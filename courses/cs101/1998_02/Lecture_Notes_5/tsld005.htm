The java.lang Package

  • Automatically imported into all programs

  • Classes corresponding to primitive data types: Boolean, Character. Number, Byte, Short, Integer, Long, Float, Double

  • Utilities: Math, System, Thread, etc.

  • Exceptions, 23 of them. Divide-by-zero, etc.

  • Errors, 23 of them, too. JVM problems.
