Summary: Loading a Class File

  • A class file has to be loaded before it can be instantiated. Happens automatically the first time it is instantiated.

  • A class file that has been loaded does not have to be instantiated at all. Example: The class that contains the main() method of an application.

  • Once a class file has been loaded, it can be instantiated any number of times without reloading.
