Packages

  • All the class files in the same directory are called a package.

  • Example: the java.lang package. Each .class file in the package is in a directory named …\java\lang (…/java/lang on Unix).

  • The CLASSPATH environment variable tells where “ … ” is on a particular system.

  • Java can find directories and .class files that are inside .zip files easily.
