Sum the odd columns of a doubly-subscripted array:

  public static int sumOdd( int[][] x ) {
  int sum = 0;
    for ( int i = 0; i < x.length; i++ )
      for ( int j = 1; j < x[ i ].length; j += 2 )
        sum += x[ i ][ j ];
    return sum;
    }
(Whether the method actually needs to be public or static is a point we will be discussing in class.)

Return true if a doubly-subscripted array is a square matrix:

  boolean isSquare( int[][] a ) {
    for ( int row = 0; row < a.length; row++ ) {
      if ( a[row].length != a.length ) {
        return false;
        }
      return true;
      }
    }
