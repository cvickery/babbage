Tour() Algorithm

  • Set length to 1

  • Calculate row and col from starting square

  • while ( true )

    • solution[row][col] = length
    • set minAccess to any value > 8
    • loop through 8 possible moves
      • If move is on-board and not to an already-visited square, decrement the corresponding accessibility by one. If new accessibility is less than minAccess, replace minAccess.
    • If minAccess is still > 8, no move can be made, so return.
    • Add one to length.
