Composition and Inheritance

  • Composition: Instances of one class contain instances of other classes.

    • Example: a Tour contains an array.
    • “Has a” relationship. “A Tour has a solution array.”

  • Inheritance: One class is derived from another class.

    • Example: HTour is derived from Tour.
    • “Is a” relationship. “An HTour is a kind of Tour.”
