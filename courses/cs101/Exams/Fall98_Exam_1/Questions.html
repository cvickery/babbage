Part I Short Answers

The next five questions are based on the following code.

public class Exam {

static Item xx;

  public static void main(String[] args) {

    for (int i=0; i<1000; i++) {

      xx = new Item();

      }

    }

  }

class Item {

  static final int BIG = 5;

  private int data = 72;

  // other code might go here

  }

2. How many variables named BIG will be created?

3. How many variables named data will be created?

Part II Write Code

Write the following code on separate pages of your answer sheets.

Code 1. Write a complete Java application named Summit that reads lines from the user until the user types Control-Z. Each line contains an arbitrary set of tokens separated by spaces. For each token that is not a number, the program prints an error message and continues to process the user’s input. Before exiting, the program prints the numeric sum of all the tokens that were valid numeric strings.

Code 2. Write a method named beLittle that receives a two-dimensional array of char, and returns a double that tells the proportion of elements of the array that were lower case. Use the following method, defined in class Character, to test the characters:

public static boolean isLowerCase(char x);

Code 3. Write a complete Java Application named echo that does nothing but print all its command line arguments, one per line. If there are no command line arguments, it prints "No command line arguments."
