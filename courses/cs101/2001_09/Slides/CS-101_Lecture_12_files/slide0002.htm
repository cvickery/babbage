• Class Hierarchy
– Hello (abstract)
• HelloGUI
• HelloCL
• HelloBR
– Hello has
• A protected String, msg
• A toString() method that returns msg.
• The main() method for the application.
– HelloGUI has a no-arg constructor that gets the user’s name using
JOptionPane, and builds a String for msg.
– HelloCL has a constructor that receives an array of Strings, which
it concatenates with spaces to fill in the user’s name and initialize
the msg.
– HelloCL has a no-arg constructor that reads the user’s name from
the command line, and initializes msg.
