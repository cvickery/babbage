Throws, and try/catch/finally
• A method that throws an exception or which calls
another method which throws an exception for
which there is no try/catch must declare that fact:
 public static void main( String[] args) throws
IOException { … }
• try/catch/finally
 try { … }
 catch ( IOException ioe ) { A }     // subclass
 catch ( Exception e ) { B }     //superclass
 finally { C }
– A, B, and C are possibly empty sequences of
statements.
