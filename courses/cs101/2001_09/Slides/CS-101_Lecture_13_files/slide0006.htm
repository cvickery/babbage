You Can Create Your Own
Exception Classes
 public class HelloException extends Exception
 {
   public HelloException( String str )
   {
      super( str );
   }
 }
• The reason for doing so is that the name of the exception class helps
document what went wrong.
– HelloException objects would be thrown when something went wrong
with Hello operations, just as IOException objects get thrown when
something goes wrong with I/O operations and
ArrayIndexOutOfBoundsException objects are thrown when an array
index (subscript) value is too large, etc.
