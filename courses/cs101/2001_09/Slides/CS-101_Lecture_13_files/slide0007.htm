Class java.io.File
• Represents a file or directory pathname
– Sep Dir Sep Dir Sep Dir Sep Fil
• Sep is system separator: \ in Windows, / in Unix
• Dir is the name of a directory
• Fil is the name of a file
– May be absolute (starting at the root) or relative
(starting, usually, in the current directory).
• Methods
– isFile(), isDirectory(), canRead(), canWrite(),
renameTo(), delete(), getName(), getParent(),
getCanonicalPath()
– Many more.
