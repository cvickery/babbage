• The this parameter for instance methods
– The difference between static methods and instance
methods is that an extra parameter, a reference to an
object, is always passed to instance methods but never
to static methods.
– The extra parameter may be used inside the instance
method by the special variable name, this.
– The this parameter is what associates a method
invocation with a particular object.
