Review Material
The this parameter for instance methods
The difference between static methods and instance methods is that an extra parameter, a reference to an object, is always passed to instance methods but never to static methods.
The extra parameter may be used inside the instance method by the special variable name, this.
The this parameter is what associates a method invocation with a particular object.
The new operator
Creates and initializes a new object.

What new Does
Make sure the class has been loaded.
Allocate memory for an object from the heap
Initialize the memory to binary zeros.
Primitive types will be zero, false, or ‘\0’; references will be null.
Notice that this applies to instance variables, not local variables, which are not initialized in any way, and not static variables, which were initialized when the class was loaded.

Access Modifiers
Access Modifiers are: public, private, or, protected.
The modifiers affect the scope of a field or method.
If omitted, a field or method has “package scope.”
Classes (and interfaces) can only be marked public.
If not marked, they too have “package scope.”
Protected has to do with inheritance, a concept of object oriented (as opposed to object based) programming
