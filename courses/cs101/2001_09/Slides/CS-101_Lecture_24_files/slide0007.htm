Center a Window
• Build the user interface and pack() it.
• Before calling show()
– Get the size of the user’s screen.
– Get the size of the window.
– Call setLocation() to position the window
• x = half the width of the screen minus half the width of the
window.
• y = half the height of the screen minus half the height of the
window.
