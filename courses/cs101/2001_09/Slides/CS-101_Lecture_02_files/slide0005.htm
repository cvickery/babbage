The jar Command
• Can be used to manage “zip” files.
– Jar is a free alternative to WinZip that comes with the
JDK.  It’s a command, just like javac and java.
– Jar files conventionally have names ending in “.jar,”
but you can use “.zip” instead if you want to
– Jar files automatically include some additional files that
zip files don’t have  (Manifest, etc.), but they can be
ignored.
– JVMs know how to extract class files from jar files
automatically, but for this course you will put just .java
files inside the jar files you submit to me.
