HelloCL.java
• Like HelloGUI.java, but the user’s name is taken
from the Java command line.
–  public static void main( String[] args ) …
•  args is an array of strings, one for each word typed on the
command line by the user.
• The elements of an array are accessed by an integer subscript.
The variable names i, j, k are conventional names for
subscripts, but any variables can be used.
• The number of elements in an array is given by the value of the
field named length.
• The first element of an array has a subscript value of zero.
•  To print each command line argument, if there are any:
     for (int i = 0; i < args.length; i++)
          System.out.print( “ “ + args[i] );
