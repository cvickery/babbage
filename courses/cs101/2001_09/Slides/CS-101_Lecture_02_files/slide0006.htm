Jar Command Options
• Create a jar file
–  jar cvf Assignment_2.jar *.java
• Creates a jar file named Assignment_2 that contains all the files that
have names ending in .java in it.
• List the contents of a jar file
–  jar tvf Assignment_2.jar
• Lists all the files in the jar file named Assignment_2.jar
• Extract files from a jar file
–  jar xvf Assignment_2.jar
• Extracts all the files from Assignment_2.jar into the current directory.
• I will use this command when grading your assignments.
• See the online documentation for more details about the
options.
