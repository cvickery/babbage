Event Handling Terminology
• AWT
– Abstract Windowing Toolkit.  The Java classes that
interact with the operating system to display windows
on the screen and to respond to user interactions.  First
graphical operation starts a separate thread of execution
that the AWT uses to respond to the user’s actions.
• Event
– When a user interacts with a component, the AWT
thread creates an Event object (or a subclass of Event),
which has instance variables giving information about
the event.
• What component the user interacted with
• What mouse button was clicked and/or what key was pressed
• Mouse coordinates, etc.
