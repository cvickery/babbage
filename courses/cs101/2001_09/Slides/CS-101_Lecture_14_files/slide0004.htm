Signed and Unsigned Numbers
• Unsigned: Values from 0 to 2n-1
– Where n is 8, 16, 32, 64 for byte, short, int, long
• Signed: Values from –2n-1 to + 2n-1-1
– Using Two’s Complement Representation
– Same as unsigned binary, except leftmost bit has a negative
weight. Sample Code to convert Strings to int.
• Sign Extension Rule
– For example, the value 11012 (-3) can be extended to 8 bits by
prepending 4 copies of the leftmost bit: 111111012
• Java doesn’t have unsigned data types, but class
DataInputStream has a readUnsignedByte() method.
– Returns an int with the leftmost 24 bits set to zero.
