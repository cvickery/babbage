Selecting A File
• JFileChooser
– Constructor takes initial pathname, which defaults to
“user’s home directory”
• On Windows 9x, this defaults to the C:\Windows directory!
• Use “.” instead.
– Remembers last directory user navigated to, so don’t
create new ones.
