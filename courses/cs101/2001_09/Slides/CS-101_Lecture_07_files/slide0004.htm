Packages
• Used to gather a set of related classes in one
directory or jar file.
– Sometimes called a “class library.”
• The Default Package
– All the class files in the “current” directory.
• To put a class into a package, put a package
statement at the very beginning of the .java file
–  package edu.queens.vickery;
– Compile the .java file(s) using the –d option
• At run time, the JVM searches the “standard
places” for class files referenced by the program
