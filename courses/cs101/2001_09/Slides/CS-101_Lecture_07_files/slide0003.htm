Constructors
• Compiler provides default no-arg constructor if none is
defined in the class.
– Does nothing.
• Class author can provide multiple constructors.
– “Constructor Overloading”
– Each must have a different set of arguments.
– No-arg constructor not provided if the class contains any
constructors.
• Constructors can be public, private, or protected
– Private constructors can be invoked as this() as the first statement
of other constructors.
• Used to perform common initialization for other constructors.
– A private no-arg constructor can be used to make a class un-
instantiable.
