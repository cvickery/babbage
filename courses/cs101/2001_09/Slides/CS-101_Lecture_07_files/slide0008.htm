Inheritance
• Every class extends some other class.
–  class HelloCL extends java.lang.Object;
– If a class directly extends java.lang.Object, that doesn’t
have to be declared.  Otherwise, the direct superclass
must be declared.
• The direct superclass is sometimes called the base
class.
• Subclasses are “bigger” than superclasses because
they add fields and/or methods to the classes from
which they are derived.
