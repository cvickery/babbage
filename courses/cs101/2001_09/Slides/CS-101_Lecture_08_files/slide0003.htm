Inheritance vs. Composition
(“is-a” vs. “has-a”)
• If class Y extends class X
– Objects of class Y have all the properties of objects of class X plus
whatever additional properties class Y provides.
– A Y is a X (and more).
• If class A contains either an instance or static field of class
B
– Objects of class A are composed of objects of class B, among other
things.
– An A has a B (and more).
• Inheritance and Composition are two totally independent
concepts.
