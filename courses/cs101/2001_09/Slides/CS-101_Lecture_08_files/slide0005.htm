• Class P extends Object, implements methodP() but
doesn’t override Object.toString()
• Class Q extends P, implements methodQ(), and
overrides toString()
•  o, p, and q are references variables of type Object,
P, and Q.
–  q is-a Q, P, and Object
–  p is-a P and Object
–  o = p; and o = q; are always safe.
–  p = q; is always safe.
–  q = (Q) p; and q = (Q) o; might be safe.
–  p = (P) o; might be safe
