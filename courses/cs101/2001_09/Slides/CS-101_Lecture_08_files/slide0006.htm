Invoking Methods
1. If the invocation looks like xxx.method();
– If xxx is a class name, the method has to be static.
– If xxx is an object reference, the method has to be non-static.
• This form can be used for static methods too, but it’s use is
discouraged.
2. If the invocation looks like method();
– The method must be defined in the same class as the method
from which the invocation is being made.
• If the method is an instance method, this is the same as
this.method();
• If the method is a static method, this is the same as
Classname.method();
3. If the invocation looks like methodA().methodB();
–  methodA is invoked using rule 2, above
– methodA must return an object reference to use for invoking
methodB, as in rule 1 above.
