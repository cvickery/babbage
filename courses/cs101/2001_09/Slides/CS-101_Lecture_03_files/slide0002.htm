Computers and JVMs
• Computer: Processor, Memory, I/O
– Pentium is an example of a processor (CPU), but there are many
others, each with its own machine language
– “Memory” is often called “RAM.”
• Holds programs as they are executed by the processor.
• Volatile: information lost when electricity goes off.
– I/O includes disks, keyboard, mouse, monitor
• Java Virtual Machine (JVM)
– A program that simulates the behavior of a particular processor.
The machine language is called Java bytecode.
– Classfiles contain the Java bytecode generated by a Java compiler.
– It would be faster to translate Java directly into a real processor’s
machine language, but not as portable.
