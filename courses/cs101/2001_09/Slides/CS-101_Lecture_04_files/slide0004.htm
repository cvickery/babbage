Loading/Instantiating a Class
• When a class is first referenced at run time, all its methods,
both static and non-static, are loaded from the class file
into an area of memory called the Method Area.
• When a class is loaded into memory, a single copy of each
static variable (“attribute”) of the class into the Method
Area.
• When a class is instantiated, which is done using the new
operator, memory for all the class’s instance variables is
allocated from from an area of memory called the Heap.
– This memory is “the object” created by new.
– There is a pointer from the object back to the class’ memory in the
Method Area.
