Constant Pool Entry Structure
• Each entry in the Constant Pool section of a class
file starts with a one-byte tag that tells what kind
of entry it is, followed by the information for the
particular type of entry.
 int tag = dis.readUnsignedByte();
 switch ( tag )
 {
   case CONSTANT_Integer:
     CONSTANT_Integer_info = dis.readInt();
     break;
   case CONSTANT_Utf8:
     length = 0x0000FFFF & dis.readShort();
     CONSTANT_Utf8_info = new byte[ length ];
     for( int i = 0; i < length; i++ )
       CONSTANT_Utf8_Info[i] = dis.readByte();
