Development Environment
Editor
Recommended: vim and gvim.
Free from vim.org
Runs on Windows and Unix
Already available on forbin
Provides syntax highlighting, automatic identation, automatic pair matching, and many advanced features you don’t have to worry about unless you want to.
Compiler (javac) and JVM (java)
Free from Sun.  Use JDK version 1.3 or later.
Jikes, free from IBM, is a faster compiler.
Do not used notepad and/or an IDE such as JBuilder.
Code formatting counts, and these environments will not give you the control and productivity you need.
You can profitably use an IDE after the course is over.

Doing an Assignment

Invoke editor, compiler, and JVM
Test program carefully before submitting it!
