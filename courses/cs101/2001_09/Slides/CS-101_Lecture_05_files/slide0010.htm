Getting from References to
Objects to Classes
• Every object’s memory includes a pointer to the
class information in the method region of memory.
– Instance methods are located from an object reference
by following that pointer.
• Instance methods are passed an extra, hidden,
parameter that is a copy of the reference variable
used to access the method.  Inside the method, that
pointer can be accessed using the variable name
“this”.
– The compiler automatically accesses the instance
variables referenced in a non-static method by using the
this pointer to locate the object in memory.
