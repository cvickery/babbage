toString() and Accessor Methods
Class ClassFile provides a toString() method as specified in the Assignment.

Class ConstantPoolEntry provides a toString() method that returns a describing the constant.  The format depends on the type of constant, so there will be a big switch statement in there.
Class ClassFile provides getXXX() methods for each field in the class file.
