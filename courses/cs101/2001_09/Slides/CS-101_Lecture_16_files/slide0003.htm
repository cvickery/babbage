• ‘&’ is the “bit-wise AND” operator
– Can be used to force bits to be zero
Ø 0 & 0 = 0
Ø 0 & 1 = 0
Ø 1 & 0 = 0
Ø 1 & 1 = 1
• ‘|’ is the bit-wise OR operator
• Can be used to force bits to be one
Ø 0 | 0 = 0
Ø 0 | 1 = 1
Ø 1 | 0 = 1
Ø 1 | 1 = 1
