CS-081 Assignment 8

Introduction

For this assignment you are to format your list of navigation links so it looks and operates somewhat like the horizontal one designed in class.

Description

As a minimum, create a background image for the links with a fixed width (such as 150px) with rounded corners on the top. Make sure the anchors fill the list items so the text in the anchors is centered in the images. Use margins and padding to make the list look as much as you can like the links are tabs on the edges of file folders.

Optional: Use a different background image for links that are being hovered over and for the link on the "current page".

Optional: Use the sliding doors technique so the imges stay properly formed when the user makes text bigger or smaller.
