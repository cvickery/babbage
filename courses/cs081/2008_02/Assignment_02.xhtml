CS-081 Assignment 2

Introduction

Use Dreamweaver to set up your web site for the course.

The Assignment

Instructions for accessing your lab account are at the TREE Lab Information web page. The assignment for February 14 is to log into your account, to set up the local part of your course web site as demonstrated in class on February 7, and to create an index page for your site based on a template file that I am providing.

Get the template.xhtml file from the Zip file linked to from the course schedule (template.zip). Download the zip file, and extract template.xhtml into your My Website directory. The, from within Dreamweaver, duplicate the template (type Control-D with the file selected), then rename Copy of template.xhtml to index.xhtml. (If you double-click on the file name, it will open the file for editing, but if you click twice with a pause between the two clicks the second click will let you edit the file name.) Now you should be able to view the template simply by using the URL, http://hostname.cs.qc.cuny.edu/~accountname/, where hostname is the name of the computer you are logged into, and accountname is your own account name. Note that the character before your account name is a tilde (upper left corner of the keyboard), not a dash.

We will go over questions about the assignment in class on February 14; you will be able to submit this assignment up to one week late for full credit, but I will be giving out another assignment on the 14th, and that one will not be accepted late.

Submit The Assignment

When your site is set up and you have tested it, be sure you log off the lab computer so your work will be copied to the server. Then send me an email message telling me your assignment is ready, and I will copy it to my computer (babbage) from the server, and check it out. My email address is:

Christopher.VickeryATqc.cuny.edu

Be sure to sign your email so I know who sent it!
